# Arxiv Research Assistant  
A Multi-Agent Sequential AI model built with Google ADK that fetches the latest research papers and summarizes them into one file

<img width="1125" height="728" alt="image" src="https://github.com/user-attachments/assets/8d5f035a-f76f-4c78-ab48-45061e9e7310" />


## Setup:  
create a virtual environment  
`python -m venv venv`  

Install the packages:  
`pip install -r requirements.txt`

## Run the script:  
`python start.py`
